# Internal Simulations

This folder contains simulation-based validation for the `EBSmoothr` package.

The main artifact in the current folder is an internal validation run for the
exact Matern backend.

## Files

- `run_matern_validation.R`: reproducible simulation script.
- `study_iid_vs_matern_selection.R`: staged iid-vs-Matern model-selection
  experiment under iid-normal truth.
- `study_2d_matern_vs_iid_comparison.R`: lightweight 2D comparison between
  exact Matern smoothing and an iid normal prior under smooth and iid truths,
  with an optional PC-prior mode for the Matern fit.
- `results/`: saved tables, figures, and markdown summary generated by the
  script.
